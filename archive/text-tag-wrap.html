css文本尾部标签跟随文本自动换行

Baomer's Blog
2021-04-15
阅读量: 0
css

在开发中，经常会遇到在title后加一个tag，在title文本过多换行时，要求tag在文本末尾也跟着换行。

看起来很简单的样式，其实也不太好实现。

借助Array.prototype.split('')将title字段没个字都切割出来为一个数组。再遍历数组将没个字单独放在一个div中，tag在所有div之后。最后对父元素做flex布局。

display: flex;
justify-content: flex-start;
align-items: center;
flex-wrap: wrap;
